Uses

The stack behind OSIG.

OSIG is built with simple infrastructure and tools that are easy to operate across multiple small projects.

Product

  • OSIG: generated Open Graph images.
  • Statushen: public status page.
  • Buttondown: newsletter delivery.

Application

  • Django: primary web framework.
  • Django-Q2: background tasks and scheduled work.
  • StimulusJS: focused frontend interactivity.
  • Replicate: AI model deployment.

Operations

  • Hetzner: cloud infrastructure.
  • CapRover: deployment management.
  • PostgreSQL: relational database.
  • Redis: cache and message broker.
  • Sentry: error tracking.
  • OpenTelemetry: observability.
